# TNT Bulldozer Plugin

Плагин для Minecraft 1.16.5 (Spigot/Paper), добавляющий уникальное зачарование для кирок.

## Описание

**TNT Бульдозер** - административное зачарование I уровня для кирок, которое при добыче блока создает мгновенный "взрыв" в области 3x3x3, разрушающий все обычные блоки.

## Особенности

- ✅ Область эффекта: куб 3x3x3 с центром в разрушаемом блоке
- ✅ Совместимо со всеми зачарованиями (Шелковое касание, Удача, Эффективность)
- ✅ Разрушает только блоки с низкой прочностью (≤ 3.0)
- ✅ Защищает индустриальные блоки (бедрок, командные блоки, эндер-сундуки)
- ✅ Интеграция с WorldGuard
- ✅ Визуальные эффекты (частицы взрыва)
- ✅ Звуковые эффекты (звук взрыва ТНТ)
- ✅ Инструмент теряет только 1 единицу прочности за активацию

## Установка

1. Скомпилируйте плагин: `mvn clean package`
2. Скопируйте `TNTBulldozer-1.0.0.jar` из папки `target` в папку `plugins` сервера
3. Перезапустите сервер
4. Настройте `config.yml` при необходимости

## Команды

Зачарование не имеет собственных команд. Используйте стандартные команды Minecraft:

```
/give @p diamond_pickaxe{Enchantments:[{id:"minecraft:tnt_bulldozer",lvl:1}]}
/enchant @p minecraft:tnt_bulldozer 1
```

## Конфигурация

```yaml
tnt-bulldozer:
  display-name: "&6TNT Бульдозер"
  radius: 1  # 1 = 3x3x3, 2 = 5x5x5
  enable-particles: true
  enable-sound: true
  enable-camera-shake: false
  compatible-with-fortune: true
  compatible-with-silk-touch: true
```

## Зависимости

- Spigot/Paper 1.16.5
- WorldGuard 7.0.5+ (опционально)
